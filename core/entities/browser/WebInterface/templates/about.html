BookManager description

Non-useful app if you have cracked (with "Koreader") Kindle or other e-book and want to save your books.

Book database (Book_db)

Inner nosql database for interaction in interactive or non-interactive mode.

Methods:

  • CRUD operations (create, read, update, delete)
  • Inner syntax interpreter for interactive mode

Modes:

  • Interactive - manual BQL command input
  • Non-interactive - auto mode, select commands

Kindle history

Module used for interaction with read file.

Methods:

  • Convenient way to save books that you want
  • Delete books that you want
  • Write history about your already read books

Modes:

  • Auto mode - automatically save books in file
  • Manual mode - ask user about anything

Transfer book

Module for transferring books from your local personal computer to e-book by Custom device interface.

Book analytic

Module for analytic.
